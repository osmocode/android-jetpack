# Etiquette

## Description

mots-clé<br>
format `-` = dépense<br>
format `+` = revenus<br>
format `=` = compte sur lequel à lieu la transaction

## Sub use-case

L'utilisateur peut :

- [ ] rentrer des mots-clé pour mieux classer la transaction
- [ ] associer un format à la transaction
- [ ] ajouter/modifier des étiquettes
